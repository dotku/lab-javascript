# Arrow Function VS Regular Function

| feature | Arrow | Regular |
| --- | --- | --- |
| return default value with 'undefined' | Yes | Yes |
| arguments | No | Yes |
| this | No | Yes |
| constractor | No | Yes |
| duplicate check | Yes | No |
| hoisting | No | Yes |
