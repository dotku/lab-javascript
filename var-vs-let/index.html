Var vs Let

`var` 和 `let` 通常没有什么不同，不过在 loop 中，这两个就有不一样的效果了。

<script>
  'use strict'; // 使用 let 必须用 use strict 模式
  for( let tuce = 0; tuce < 5; tuce++ ) {
    //tuce is only visible in here (and in the for() parentheses)
  };
  console.log(tuce); // tuce is not defined
  for( var tuce = 0; tuce < 5; tuce++ ) {
    //tuce is only visible in here (and in the for() parentheses)
  };
  console.log(tuce); // 5
</script>

通常情况下，let 都是 let ... in 和 let ... of 这样的组合，let-in 用于 Object，let-of 用于 Array。
